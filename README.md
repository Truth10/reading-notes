## 📝 如何使用

### 添加新的读书笔记

1. 根据书籍类型，在对应的分类文件夹下创建 Markdown 文件
2. 使用 `templates/book-note-template.md` 作为模板
3. 填写书籍信息、核心观点、个人思考等内容
4. 在本 README 中添加书籍索引链接

### 笔记结构

每本书的笔记包含：

- 📖 **基本信息** - 书名、作者、出版信息等
- � **章节结构性精华总结** - 核心命题、解决的问题、观点拆解
- ✨ **金句摘录** - 打动人心的句子
- 🤔 **我的思考** - 个人感悟、启发、实践应用
- ⭐ **我的评价** - 评分、推荐理由、适合人群
